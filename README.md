# Xoriva — Full-Stack E-Commerce Platform

Xoriva is a full-stack e-commerce platform along with a built-in peer-to-peer thrift 
marketplace, where users can shop new products and list or buy second-hand items directly.

## 🔗 Links
- **Live:** https://xoriva.vercel.app
- **Frontend Repo:** https://github.com/Sharuna-mk/Xoriva
- **Backend Repo:** https://github.com/Sharuna-mk/XORIVA-backend

##  Tech Stack
**Frontend:** React.js, Redux Toolkit, Tailwind CSS, Axios, Figma, CSS3, Firebase  
**Backend:** Node.js, Express.js, MongoDB, JWT, Bcrypt.js, Nodemailer, Stripe, Google Gemini API
**Database** MongoDB(noSql)

##  Features

### 👤 User
- Browse, search, and purchase products
- Secure checkout via Stripe payment gateway
- List, manage, and buy second-hand items on the Thrift Marketplace along with new products
- Real-time support via Google Gemini AI Chatbot
- Account confirmation emails via Nodemailer
- JWT-based authentication and protected routes

### 🛡 Admin
- Inventory management — add, update, remove products
- Order tracking and management
- Analytics dashboard
- User management

##  Highlights
- Peer-to-peer Thrift Marketplace for second-hand listings
- Google Gemini AI Chatbot for real-time user assistance
- Redux Toolkit state optimization — reduced redundant API calls by ~35%
- Secure REST APIs with JWT authentication and Bcrypt.js password hashing
